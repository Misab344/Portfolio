What I've Built

🥈 CISPA European AI Hackathon 2025/26
01 —

Stolen Model Detection

Built an AI security system to detect stolen machine learning models. Given white-box access to a target ResNet-18 CIFAR-100 classifier and 360 suspect models, developed a pipeline assigning continuous stealing confidence scores using model fingerprinting and weight similarity analysis. Evaluated using TPR@FPR=5%.

Python Machine Learning Model Security ResNet-18
🥈 CISPA European AI Hackathon 2025/26
02 —

Watermark Forgery Attack

Implemented a watermark forgery system under near black-box conditions. Extracted invisible watermarks from 8 unknown schemes across 200 images and forged them onto clean targets. Achieved a final score of 0.7/1.0 evaluated on Detection Strength (bit accuracy) and Visual Quality (LPIPS).

Image Processing Computer Vision Python LPIPS
03 —

GPU-Accelerated Video Pipeline

Designed a high-performance parallel video processing pipeline using OpenCL and GPU programming. Implemented a multi-stage transformation: RGB to grayscale conversion, Sobel edge detection, and enhanced color reconstruction — all parallelized across GPU threads for significant performance gains over CPU processing.

OpenCL GPU Programming Parallel Computing Sobel
